# jlibc | Bare-Metal Linux Runtime
A personal sandbox for low-level exploration. This is a growing collection of bare-metal C code, custom utilities, and system experiments built completely from scratch to understand exactly how x86_64 Linux works under the hood.

Rather than just being a replacement library, this codebase serves as the raw building blocks for implementing broader software. The ultimate goal is to build applications from the absolute bottom up—analyzing and exploring systems with no hidden abstraction layers getting in the way.


Environment: **GCC 13.3.0** | **Linux 6.6.87.2 (WSL2)**
